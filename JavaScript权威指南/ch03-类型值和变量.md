# 第3章 类型、值和变量

## 类型分类

- **原始类型**(7种)：number, string, boolean, null, undefined, symbol, bigint
- **对象类型**：普通对象、数组、Set、Map、Date、RegExp、函数、类等
- 原始类型不可变(immutable)，对象类型可变(mutable)

## Number

- 64位 IEEE 754 浮点数，范围 `±1.8e308`
- 安全整数范围：`±2^53 - 1`（`Number.MIN_SAFE_INTEGER` 到 `Number.MAX_SAFE_INTEGER`）
- 特殊值：`Infinity`、`-Infinity`、`NaN`、`+0`、`-0`
- `NaN !== NaN`，检测用 `Number.isNaN()` 或 `x !== x`
- BigInt（ES2020）：任意精度整数，后缀 `n`（如 `1234n`），不能与 Number 混用

## String

- 不可变，UTF-16 编码，索引从 0 开始
- 三种写法：`"..."`、`'...'`、`` `...` ``
- 模板字面量：支持多行和 `${expr}` 插值
- 转义序列：`\n`、`\t`、`\\`、`\uXXXX`、`\xXX`

## Boolean

- 假值(falsy)：`false`、`null`、`undefined`、`0`、`-0`、`NaN`、`""`
- 其余所有值均为真值(truthy)

## null vs undefined

- `null`：表示"空值"，`typeof null === "object"`（历史遗留bug）
- `undefined`：未初始化变量的默认值，函数无返回值时返回的值
- `null == undefined` → `true`（宽松相等），`null === undefined` → `false`
- 两者都没有属性或方法

## Symbol（ES6）

- 创建唯一标识符：`Symbol("desc")`
- `Symbol.for("key")`：全局符号注册表，可跨 realm 共享
- `Symbol.keyFor(sym)`：获取全局符号的 key
- Well-known Symbols：`Symbol.iterator`、`Symbol.toPrimitive`、`Symbol.hasInstance` 等

## 全局对象

- 浏览器：`window`、`self`、`frames`、`globalThis`（ES2020）
- Node.js：`global`、`globalThis`

## 类型转换

- 显式转换：`Boolean()`、`Number()`、`String()`、`BigInt()`、`Symbol()`
- 隐式转换触发场景：`+` 运算符、`==` 比较、`if()` 条件、逻辑运算符
- 对象→原始值：优先 `Symbol.toPrimitive`，其次 `valueOf()`，最后 `toString()`
- `"+"` 运算符：有字符串则连接，否则数值加
- `"=="` 宽松相等算法（允许类型转换），`"==="` 严格相等（类型不同直接false）

## 变量声明

| 关键字 | 作用域 | 可重复声明 | 可重新赋值 | 存在TDZ | 全局对象属性 |
|--------|--------|-----------|-----------|---------|------------|
| var | 函数 | 是 | 是 | 否 | 是 |
| let | 块 | 否 | 是 | 是 | 否 |
| const | 块 | 否 | 否 | 是 | 否 |
| function | 函数(可提升) | - | - | - | - |
| class | 块 | 否 | 否 | 是 | 否 |

### 解构赋值
```js
let [a, b] = [1, 2];           // 数组解构
let {x, y} = {x: 1, y: 2};     // 对象解构
let {a: alias} = {a: 1};       // 重命名
let [first, ...rest] = array;  // 剩余元素
```

## 容易弄错的点

1. **`typeof null === "object"`** — 历史遗留 bug，用 `x === null` 检测 null

2. **浮点数精度丢失**：`0.1 + 0.2 !== 0.3`，用 `Number.EPSILON` 或 `toFixed`
   ```js
   0.1 + 0.2 === 0.3                        // false!
   Math.abs(0.1 + 0.2 - 0.3) < Number.EPSILON  // true
   ```

3. **`NaN` 是唯一不等于自身的值**：`NaN === NaN` → `false`。用 `Number.isNaN()` 而非全局 `isNaN()`
   ```js
   isNaN("hello")         // true（先转为 NaN 再判断，误导！）
   Number.isNaN("hello")  // false（正确，hello 不是 NaN）
   ```

4. **`==` 的隐式转换规则复杂**：`"" == false` → `true`，`" " == false` → `true`。永远优先用 `===`

5. **BigInt 不能与 Number 混用**：`1n + 1` → TypeError。Math 方法不接受 BigInt

6. **`const` 不可重新赋值，但对象属性可变**：`const obj = {}; obj.x = 1;` 合法。若需冻结用 `Object.freeze()`

7. **TDZ 陷阱**：`typeof` 对未声明变量返回 `"undefined"`，但 TDZ 中的 `let/const` 用 `typeof` 直接抛 ReferenceError

8. **`Symbol.for("x")` 可在不同 iframe 间共享同一符号**，而 `Symbol("x")` 每次创建新的唯一符号
