# 第7章 数组

## 创建数组

```js
let a = [1, 2, 3];              // 字面量
let b = new Array(10);           // 指定长度（参数为单个数值时）
let c = Array.of(1, 2, 3);       // ES6，避免歧义
let d = Array.from(iterable);    // 从可迭代对象或类数组创建
let e = [...iterable];           // 展开运算符
```

## 读写与稀疏数组

```js
a[0]         // 索引访问
a.length     // 元素个数，可写（增大创建空位，减小截断）
a.length = 0 // 清空数组
```
- 稀疏数组：length > 实际元素数，中间有空位（hole）
- 空位行为和 `undefined` 不同（forEach/find等跳过空位）

## 栈和队列方法

| 方法 | 操作 | 返回值 | 改变原数组 |
|------|------|--------|-----------|
| `push(...items)` | 末尾添加 | 新 length | 是 |
| `pop()` | 末尾移除 | 移除的元素 | 是 |
| `unshift(...items)` | 开头添加 | 新 length | 是 |
| `shift()` | 开头移除 | 移除的元素 | 是 |

## 子数组操作

| 方法 | 描述 | 改变原数组 |
|------|------|-----------|
| `slice(start, end)` | 浅拷贝片段 | 否 |
| `splice(start, deleteCount, ...items)` | 删除/插入 | 是 |
| `fill(value, start, end)` | 填充 | 是 |
| `copyWithin(target, start, end)` | 内部复制 | 是 |

## 搜索与排序

```js
a.indexOf(item)        // 严格相等查找，返回索引或 -1
a.lastIndexOf(item)
a.includes(item)       // ES6，返回布尔
a.find(fn)             // ES6，返回第一个满足条件的元素
a.findIndex(fn)        // ES6，返回索引
a.sort(compareFn)      // 默认按字符串排序，改变原数组
a.reverse()            // 反转，改变原数组
```

## 迭代方法

| 方法 | 返回 | 用途 |
|------|------|------|
| `forEach(fn)` | undefined | 遍历执行 |
| `map(fn)` | 新数组 | 映射转换 |
| `filter(fn)` | 新数组 | 过滤 |
| `reduce(fn, init)` | 单值 | 归约 |
| `reduceRight(fn, init)` | 单值 | 从右归约 |
| `every(fn)` | boolean | 全部满足 |
| `some(fn)` | boolean | 任一满足 |
| `flat(depth)` | 新数组 | 展平 (ES2019) |
| `flatMap(fn)` | 新数组 | map 后 flat(1) (ES2019) |
| `join(sep)` | 字符串 | 拼接 |

## 定型数组 (Typed Arrays)

- ES6，固定长度、固定类型（Int8/Uint8/Int16/Uint16/Int32/Uint32/Float32/Float64/BigInt64/Biguint64）
- 基于 ArrayBuffer，性能优于普通数组
- `subarray()` 返回共享内存的视图
- 详见第11章

## 类数组对象

- 有数值索引和 length 属性的对象
- `arguments`、DOM NodeList、字符串
- 用 `Array.from()` 或 `Array.prototype.slice.call()` 转为真正的数组

## 容易弄错的点

1. **`sort()` 默认按字符串排序**，不是按数值！这是最常见的数组 bug
   ```js
   [1, 2, 10, 20].sort()         // [1, 10, 2, 20]  ← 按字符串！
   [1, 2, 10, 20].sort((a,b)=>a-b) // [1, 2, 10, 20]  ← 按数值
   ```

2. **`new Array(3)` vs `Array.of(3)`**：单个数值参数时，前者创建长度为3的空数组，后者创建 `[3]`。`Array.of` 消除此歧义

3. **`forEach` 不能用 `break`/`return` 跳出**：`return` 只跳过当前回调（类似 continue）。需要用 `for/of` 或 `some/every`

4. **`indexOf` 用 `===` 查找，找不到 NaN**：`[NaN].indexOf(NaN)` → `-1`。用 `includes` 可以找到 NaN

5. **`splice` 改变原数组并返回删除的元素**：常被误认为返回新数组
   ```js
   let a = [1, 2, 3];
   let b = a.splice(0, 1); // b = [1], a = [2, 3]
   ```

6. **稀疏数组的坑**：`new Array(3).map(x => 1)` → 空位不被 map/forEach 遍历！结果是 `[empty × 3]`

7. **`length` 可写**：`arr.length = 0` 清空数组；设置大于当前 length 会创建空位

8. **`concat` 和 `slice` 是浅拷贝**：嵌套的数组或对象仍共享引用
