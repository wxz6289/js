# 第1章 JavaScript简介

JavaScript 是一门高级、动态、解释型编程语言，非常适合面向对象和函数式编程风格。

## 核心语言 vs 宿主环境

- **核心语言**：定义最小限度的 API（数值、文本、数组、集合、映射），不含输入/输出
- **宿主环境**：提供输入/输出及复杂特性（网络、存储、图形）。浏览器和 Node.js 是最常见的宿主

## 语言特性

- 基于原型(prototype)的面向对象
- 函数是第一类对象(first-class function)，支持函数式编程
- 动态类型，运行时检查
- 单线程事件驱动模型

## 表达式与语句

- **表达式**：计算值，不改变程序状态
- **语句**：没有值，但改变程序状态
- 函数是一块可重复调用的代码块

## 版本演进

| 版本 | 年份 | 主要特性 |
|------|------|---------|
| ES3 | 1999 | 基础标准 |
| ES5 | 2009 | 严格模式、JSON、数组方法(add) |
| ES6/ES2015 | 2015 | class、模块、箭头函数、Promise、let/const、Symbol、迭代器、生成器 |
| ES2016 | 2016 | `**` 指数运算符、Array.prototype.includes |
| ES2017 | 2017 | async/await、Object.values/entries、SharedArrayBuffer |
| ES2018 | 2018 | 异步迭代、Rest/Spread 属性、Promise.finally |
| ES2019 | 2019 | Array.flat/flatMap、Object.fromEntries、可选catch |
| ES2020 | 2020 | 可选链`?.`、空值合并`??`、BigInt、Promise.allSettled |

## 在页面中加入 JS

1. 内部 `<script>...</script>`
2. 外部 `<script src="...">`
   - `async`：异步下载，下载完立即执行，不保证顺序
   - `defer`：异步下载，HTML 解析完后按顺序执行
3. 内联事件处理器（不推荐）
